Settlement / Town Generator Plugin for Campaign Forge

Install:
1) Unzip this file.
2) Copy the folder 'settlement' into:
   campaign_forge/plugins/

Run Campaign Forge. You should see:
- "Settlement / Town Generator" in the module list.

Notes:
- Ctrl+G generates.
- "Preview Map" opens a separate window with zoom (mouse wheel) and pan (drag).
- "Export Session Pack" writes map.png, map.svg, district_key.md, locations.md, rumors.md into exports/session_packs.
